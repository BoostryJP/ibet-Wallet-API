## Start the API server
Install packages
```
$ pip install -r requirements.txt
```

You can start the API server with:
```
$ ./bin/run.sh start
```

## Running the tests
Install packages
```
$ pip install -r ./app/tests/requirements.txt
```

You can run the tests with:
```
$ pytest app/tests/
```
