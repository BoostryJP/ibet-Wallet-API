## Starting and Stopping the Server
Install packages
```
$ pip install -r requirements.txt
```

You can start (or stop) the API server with:
```
$ ./bin/run.sh start(stop)
```

## Running the tests
Install packages
```
$ pip install -r ./app/tests/requirements.txt
```

You can run the tests with:
```
$ pytest app/tests/
```
