## 環境構築
* モジュールインストール　→　テスト環境構築手順と同じ。
* 環境変数として以下の設定を行なっておく。

```
export BASIC_AUTH_USER="XXXXX"
export BASIC_AUTH_PASS="XXXXX"
```

## テスト実行手順
* locustfile.pyが存在するディレクトリまで移動する。
* 以下のコマンドで負荷をかけることができる。（コマンドラインモード）

```
$ locust -H {エンドポイントのURL} --no-web -c 10 -r 3
```

* -c: 何クライアント作成するか
* -r: クライアントの作成スピード(毎秒)
* --no-webオプションを外すとGUIから操作が可能。
