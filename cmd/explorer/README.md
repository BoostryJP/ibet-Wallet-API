# ibet-Wallet-API BC-Explorer

## Setup

### Poetry
```bash
poetry install
```

### Pip
```bash
pip install -e ./
```

## Run

### Poetry
```bash
poetry run python src/main.py --url http://localhost:5000
```

### Pip
```bash
python src/main.py --url http://localhost:5000
```
